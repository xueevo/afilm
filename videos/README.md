# Видеофайлы для AFilm Detective Game

Эта папка предназначена для хранения видеофайлов игры.

## Необходимые видео

### Трейлер
- `trailer.mp4` - вступительный трейлер игры

### Эпизод 1: Первая встреча
- `episode1_sony.mp4` - видео с Sony TRV900 (правильный ответ)
- `episode1_canon.mp4` - видео с Canon XL1
- `episode1_panasonic.mp4` - видео с Panasonic DVX100

### Эпизод 2: Интервью с актрисой
- `episode2_canon.mp4` - видео с Canon XL1 (правильный ответ)
- `episode2_sony.mp4` - видео с Sony HC1
- `episode2_canong10.mp4` - видео с Canon HF G10

### Эпизод 3: Киношная сцена
- `episode3_panasonic.mp4` - видео с Panasonic DVX100 (правильный ответ)
- `episode3_sony.mp4` - видео с Sony TRV900
- `episode3_canon.mp4` - видео с Canon XL1

### Эпизод 4: Финальная улика
- `episode4_gopro.mp4` - видео с GoPro Hero3 (правильный ответ)
- `episode4_sony.mp4` - видео с Sony HC1
- `episode4_canon.mp4` - видео с Canon HF G10

## Рекомендации по видео

- Формат: MP4 (H.264)
- Разрешение: 1280x720 или 1920x1080
- Длительность: 30-60 секунд для эпизодов, 1-2 минуты для трейлера
- Размер: не более 50 МБ на файл для быстрой загрузки

## Временные плейсхолдеры

В HTML-файлах используются временные плейсхолдеры от placeholder.com.
После добавления реальных видеофайлов они автоматически заменят плейсхолдеры.

## Как добавить видео

1. Поместите видеофайлы в эту папку (`videos/`)
2. Убедитесь, что имена файлов точно соответствуют списку выше
3. Проверьте работу сайта, открыв `index.html` в браузере
